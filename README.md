# Master-s-Thesis
Mapping Heat-Related Inequality in Berlin: Linking Socio-Economic Sensitivity and Urban Heat Exposure

Hey there :)

This repository contains the whole RStudio code used in my Master's Thesis. The thesis is a spatial overlay analysis that investigates the relationship between socio-economic heat sensitivity and heat exposure in Berlin unsing open source data. Socio-economic data was derived from the latest Germany census (Zensus 2022) unsing the highest spatial resolution available (100 x 100 m). Landsat 8/9 Collection 2 Level-2 images used to create median Land Surface Temperature rasters were accessed vis Google Earth Engine. 

Feel free to use/adapt this code if you are interested in similar topics or if it might help you in your own work.
